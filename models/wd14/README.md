# WD14 Model Files

Place WD14 ONNX files in this folder:

- `model.onnx`
- `selected_tags.csv`

The app will auto-detect these files and use WD14 for auto-tagging.
If missing, it falls back to filename-based tagging.

Recommended source:
- SmilingWolf WD14 tagger releases (ONNX + selected_tags.csv)
