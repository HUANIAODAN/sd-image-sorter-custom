# 🚀 启动脚本使用指南

## 📋 可用的启动脚本

### 1. 启动.bat (一键启动 - 中文) ⭐⭐⭐⭐⭐
- 自动检查并创建虚拟环境
- 自动安装缺失的依赖
- 自动执行数据库迁移
- 自动创建模型目录
- 启动开发服务器（支持热重载）

**使用**：双击 "启动.bat"

---

### 2. start.bat (Quick Start - 英文) ⭐⭐⭐⭐⭐
- 与 "启动.bat" 功能相同
- 英文界面

**使用**：双击 "start.bat"

---

### 3. 启动菜单.bat (高级启动菜单) ⭐⭐⭐⭐
- [1] 快速启动
- [2] 安装/更新依赖
- [3] 执行数据库迁移
- [4] 启动 (不自动重载)
- [5] 启动 (自定义端口)
- [6] 查看系统信息
- [7] 清理缓存文件

**使用**：双击 "启动菜单.bat"

---

## 🎯 推荐使用

- **新手用户**：启动.bat 或 start.bat
- **日常使用**：启动.bat 或 start.bat
- **开发调试**：启动菜单.bat
- **自定义端口**：启动菜单.bat -> [5]

---

## 🌐 启动后访问

- 前端界面：http://localhost:8000
- API 文档：http://localhost:8000/docs
- 健康检查：http://localhost:8000/api/health

---

## 🔧 常见问题

**Q: 双击没反应？**
A: 确保已安装 Python 3.8+，右键以管理员身份运行

**Q: 端口被占用？**
A: 使用启动菜单.bat -> [5] 自定义端口

**Q: 如何停止？**
A: 按 Ctrl + C

---

## 🎉 快速开始

1. 双击 `启动.bat`
2. 等待自动配置
3. 访问 `http://localhost:8000`
