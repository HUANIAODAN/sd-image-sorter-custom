# Visual Studio 主题优化指南

## 概述

此次优化将前端界面改造为完全遵循 Visual Studio Code 的浅色和深色主题风格，提供更专业、更一致的视觉体验。

## 主要改进

### 1. 配色方案

#### 浅色主题 (Light Theme)
- **背景色**: `#ffffff` - 纯白背景
- **表面色**: `#f3f3f3` - 轻微灰色用于面板
- **文字色**: `#000000` - 纯黑文字，次要文字 `#3b3b3b`
- **强调色**: `#007acc` - VS Code 标志性蓝色
- **边框色**: `#cccccc` - 柔和的分隔线
- **代码背景**: `#f5f5f5` - 代码块专用背景

#### 深色主题 (Dark Theme)
- **背景色**: `#1e1e1e` - VS Code 经典深色背景
- **表面色**: `#252526` - 面板背景色
- **文字色**: `#cccccc` - 柔和的浅色文字
- **强调色**: `#0e639c` - 深色模式下的蓝色调整
- **边框色**: `#3e3e42` - 深色边框
- **代码背景**: `#1e1e1e` - 与编辑器一致

### 2. 布局优化

- **移除卡片间隙**: 采用 VS Code 式的无间隙面板布局
- **统一边框策略**: 使用右边框分隔面板，而非四周边框
- **扁平化设计**: 移除阴影，采用线性分隔
- **紧凑间距**: 减少内边距，提高信息密度

### 3. 组件样式

#### 标签页 (Tabs)
- 从卡片式标签改为底部边框式
- 激活状态显示蓝色底部边框
- 移除背景色，使用透明背景
- 悬停时轻微背景变化

#### 按钮 (Buttons)
- 扁平化设计，无圆角
- 主要按钮使用强调色背景
- 次要按钮使用透明背景
- 禁用状态显示 50% 不透明度
- 点击时轻微位移反馈

#### 输入框 (Inputs)
- 移除圆角，使用直角边框
- 焦点时显示蓝色边框
- 悬停时边框加深
- 背景色根据主题调整

#### 徽章 (Badges)
- 从圆形改为小圆角矩形
- 更紧凑的内边距
- 统一的字体大小 (11px)
- 更好的对比度

### 4. 滚动条样式

完全模仿 VS Code 的滚动条：
- 宽度: 14px
- 轨道背景与主题匹配
- 滑块带有 3px 内边距
- 圆角滑块设计
- 悬停时颜色加深

### 5. 主题切换功能

新增主题切换按钮：
- 位于顶部导航栏右侧
- 图标式按钮设计
- 点击即可切换浅色/深色主题
- 主题设置保存到 localStorage
- 启动时自动恢复上次选择

### 6. 字体和排版

- **无衬线字体**: 使用系统原生 Segoe UI 字体栈
- **等宽字体**: Cascadia Code/Mono 用于代码显示
- **字体大小**: 13px 基础字号
- **行高**: 1.6 提高可读性
- **字体平滑**: 开启反锯齿渲染
- **字母间距**: 标题使用适当的字母间距

### 7. 视觉层次

- **标题**: 11-13px 大写字母，600 字重
- **正文**: 13px 正常字重
- **辅助文字**: 12px 灰色
- **代码**: 13px 等宽字体
- **徽章**: 11px 紧凑显示

### 8. 交互反馈

- **过渡动画**: 0.1s 快速过渡
- **悬停效果**: 背景色轻微变化
- **焦点状态**: 蓝色边框高亮
- **激活状态**: 轻微位移和颜色加深
- **选中状态**: 蓝色背景和边框

### 9. 响应式设计

- 保持原有响应式断点
- 移动端优化按钮和间距
- 平板设备自动调整布局
- 触摸友好的交互区域

## 使用说明

### 主题切换

点击右上角的主题切换按钮即可在浅色和深色主题之间切换。系统会记住你的选择。

### 自动适配

如果没有手动选择主题，系统会自动适配操作系统的主题偏好（通过 `prefers-color-scheme` 媒体查询）。

### 浏览器兼容性

- 现代浏览器全面支持
- CSS 自定义属性用于主题切换
- 滚动条样式在 Webkit 和 Firefox 上均可用

## 技术细节

### CSS 变量系统

所有颜色通过 CSS 变量定义，便于主题切换：

```css
:root {
  --bg: #ffffff;
  --text: #000000;
  --accent: #007acc;
  /* ... 更多变量 */
}

html[data-theme="dark"] {
  --bg: #1e1e1e;
  --text: #cccccc;
  --accent: #0e639c;
  /* ... 深色主题变量 */
}
```

### 主题持久化

```javascript
// 保存主题
localStorage.setItem('sd-sorter-theme', 'dark');

// 读取主题
const saved = localStorage.getItem('sd-sorter-theme');
```

### 启动时加载

主题在页面加载前应用，避免闪烁：

```html
<script>
  (() => {
    const saved = localStorage.getItem('sd-sorter-theme');
    if (saved === 'light' || saved === 'dark') {
      document.documentElement.dataset.theme = saved;
    }
  })();
</script>
```
